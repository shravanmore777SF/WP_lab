# Web Programming Lab

This repository contains all the experiments and projects completed as part of the Web Programming (WP) lab.

## Repository Structure

The repository is organized by experiment numbers:

- `exp-01/`: Basic HTML tag layout and intro image.
- `exp-02/`: CSS styling and layout.
- `exp-03/`: JavaScript basics.
- `exp-04/`: Advanced DOM manipulation.
- `exp-05/`: Event handling.
- `exp-06/`: Fetch API and async/await.
- `exp-07/`: Local Storage and state management.
- `exp-08/`: Form validation.
- `exp-09/`: React/Vite project (Step-by-step and Recitation).

## Getting Started

To run these experiments locally:

1. Clone the repository.
2. For experiments using Vite (like `exp-09`), navigate to the directory and run:
   ```bash
   npm install
   npm run dev
   ```
3. For basic HTML/CSS files, simply open `index.html` in your browser.
